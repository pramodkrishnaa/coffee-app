# Full-Stack Web Application (AI-Assisted)

## Overview
This project is a full-stack web application built to demonstrate end-to-end development skills as a fresher. The application includes dual dashboards, backend APIs, database integration, and a payment gateway.

AI tools were used responsibly to accelerate development while maintaining full technical understanding and control.

---

## Features
- Buyer Dashboard
- Seller Dashboard
- Razorpay Payment Gateway (Test Mode)
- Clean and minimal UI
- Cloud-hosted backend and database
- Live deployment with instant updates

---

## Tech Stack

### Frontend
- React
- Next.js
- Tailwind CSS
- Vite

### Backend
- Node.js
- Express.js

### Database
- Supabase (PostgreSQL)

### Payments
- Razorpay (Sandbox / Test Mode)

### Deployment
- GitHub
- Vercel

---

## Development Workflow
1. Created a PRD using ChatGPT inspired by Tiny Tiger Coffee
2. Generated base code using Lovable AI
3. Modified backend logic using Antigravity
4. Ran and optimized the app using Vite
5. Managed version control with GitHub
6. Deployed and updated live using Vercel

---

## Payment Gateway
Razorpay Test Mode is used to demonstrate payment integration without real transactions.
